# 📄 Word Counter (HTML, CSS, JavaScript)

A simple and responsive **Word Counter web app** built using **HTML, CSS, and JavaScript**. It allows users to quickly count words, characters, sentences, and paragraphs in real time.

---

## 🚀 Features

* ✅ Real-time word count
* 🔗 Check Grammar
* 🔤 Character count (with & without spaces)
* 📄 Sentence counter
* 🧾 Paragraph counter
* ⚡ Instant updates while typing
* 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the app
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Functionality and logic

---

## ⚙️ How It Works

* The app listens to user input inside a textarea.
* On every keystroke:

  * Words are counted by splitting text using spaces.
  * Characters are counted using string length.
  * Sentences are detected using punctuation (`.`, `!`, `?`).
  * Paragraphs are identified using line breaks.

---

## 💡 Example

If you type:

```
Hello world! This is a test.
```

Output will be:

* Words: 6
* Characters: 28
* Sentences: 2
* Paragraphs: 1
